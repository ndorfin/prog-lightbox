Usage examples

Thumbnail anchor


<prog-lightbox>
  <a href="./img/photo-large.jpg">
    <img src="./img/photo-thumbnail" alt="Jake the sheep frolics in the lush pasture">
  </a>
</prog-lightbox>

Text anchor (no alt text provided)

View the large photo 

<prog-lightbox>
  <a href="./img/photo-large.jpg">
    View the large photo
  </a>
</prog-lightbox>

Add alt text to dialog image

By default, prog-lightbox will use the thumbnail's alt attribute value. If no such thumbnail exists, please supply alt text for the dialog's image using the alt attribute on prog-lightbox.

View the large photo 

<prog-lightbox alt="Jake the sheep frolics in the lush pasture">
  <a href="./img/photo-large.jpg">
    View the large photo
  </a>
</prog-lightbox>
